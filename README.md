## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/yaseminyavuz/your-way-ally.git
cd your-way-ally

# Backend setup
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt

# Create .env file with your API keys
cp .env.example .env

# Run the application
python main.py
