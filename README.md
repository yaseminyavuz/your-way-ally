# Your Way Ally 🌍✈️

> AI-powered travel planning chatbot that learns from user experiences

[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.104.1-00a393.svg)](https://fastapi.tiangolo.com)

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/yaseminyavuz/your-way-ally.git
cd your-way-ally

# Backend setup
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt

# Create .env file with your API keys
cp .env.example .env

# Run the application
python main.py

Visit: http://localhost:8000/docs
✨ Features

🤖 AI Chat Assistant - GPT-powered travel recommendations
🌤️ Weather Integration - Real-time weather data
🔐 User Authentication - JWT-based security
💳 Credit System - Gamified interaction model
📚 Auto Documentation - Interactive API docs

🛠️ Tech Stack

Backend: FastAPI + SQLAlchemy + SQLite
AI: OpenAI GPT-4 integration
APIs: OpenWeatherMap, Google Places
Auth: JWT tokens with BCrypt
